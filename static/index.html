        📱 iPhone Setup
        

          

            
Step 1

            
Create a Photos album named: StreamLine Auto

          

          

            
Step 2

            
Install the iPhone Shortcut.

          

          

            
Step 3

            
After generating a Facebook post, press "Copy Photos URL".

          

          

            
Step 4

            
Run the Shortcut.

          

          

            
Step 5

            
The images will automatically be saved inside the StreamLine Auto album.

          

          

            
Shortcut Actions

            
1. Find Photos
Action: Find Photos
Filter: Album is StreamLine Auto

2. Delete Photos
Action: Delete Photos
Input: Photos

3. Get Clipboard
Action: Get Clipboard

4. Download JSON
Action: Get Contents of URL
URL: Clipboard
Method: GET

5. Parse JSON
Action: Get Dictionary from Contents of URL

6. Extract image list
Action: Get Value
Key: image_urls
Input: Dictionary

7. Repeat for every image
Action: Repeat with each item in Dictionary Value

   - Get Contents of URL
     URL: Repeat Item

   - Save to Photo Album
     Album: StreamLine Auto

8. End Repeat
Action: End Repeat

9. Finish
Action: Stop and Output
